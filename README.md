# opPtimizer

Optimization framework for AI.

## Installing

### Linux
Install
```
git clone https://github.com/virtgen/opptimizer.git
cd opptimizer
sudo ./install.sh linux
```

Uninstall
```
./install linux remove
```

### MAC

Install
```
git clone https://github.com/virtgen/opptimizer.git
cd opptimizer
sudo ./install.sh mac
```

Uninstall
```
./install mac remove
```
